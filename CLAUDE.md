# 집업 (ZipUp) — 재개발 밸류업 갤러리

## Project
재개발/리모델링 비포&애프터를 핀터레스트처럼 모아보고, 비용·시공사·밸류업 효과를 상세히 볼 수 있는 플랫폼.

## Tech Stack
- Next.js (App Router, TypeScript)
- Tailwind CSS
- Deployed to Vercel

## Key Commands
- `npm run dev` — start dev server
- `npm run build` — production build

## Design
- Pinterest-style masonry gallery
- Orange (#F97316) primary accent
- Stone color palette for neutrals
- Korean-first UI
